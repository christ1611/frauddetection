## 시스템 개요

**금융사기 신고센터**는 시민이 금융사기 피해를 직접 신고할 수 있는 웹 기반 신고 시스템입니다.  
보이스피싱, QR코드 사기, 스미싱 등 다양한 금융사기 유형에 대해 증거를 수집하고 신고서를 제출할 수 있으며, 제출된 신고 내용은 수사기관과 즉시 공유됩니다.

```
신고자  →  [금융사기 신고센터]  →  reports.txt / reports.json
                                →  금융감독원 (FSS)
                                →  경찰청 사이버수사국
```

## 주요 기능

### 🚨 신고 접수 (6단계 마법사)
| 단계 | 내용 |
|------|------|
| 1단계 | 신고자 정보 입력 (익명 신고 가능) |
| 2단계 | 사기 QR코드 카메라 스캔 또는 직접 입력 |
| 3단계 | 증거 사진 업로드 (최대 4장, 드래그 앤 드롭) |
| 4단계 | 사건 발생 위치 — GPS 자동 감지 또는 주소 직접 입력 |
| 5단계 | 사기 유형 선택 및 피해 상황 상세 기술 |
| 6단계 | 최종 검토 및 제출 — 고유 접수번호 발급 |

### 🔍 신고 조회
- 접수번호 8자리로 처리 현황 실시간 확인
- **진행 타임라인**: 접수 완료 → 검토 중 → 수사 의뢰 → 처리 완료
- 신고 내용 전체 및 첨부 사진 확인 가능

### 📊 관리자 대시보드 (`/admin`)
- 통계 카드: 총 신고 건수, 오늘 접수, 이번 달, 최다 사기 유형
- 사기 유형별 막대 차트 (실시간)
- 전체 신고 목록 — 검색, 필터, 페이지네이션
- 신고 상세 내용 펼쳐보기 및 첨부 사진 확인
- **CSV 내보내기** (한국 Excel 호환 BOM 처리)

### 🆘 긴급 연락처 버튼
- 모든 페이지 우측 하단에 고정
- 금융감독원 1332 / 경찰청 182 / KISA 118 / 금융결제원 1577-5500
- 탭하면 바로 전화 연결 (모바일)

### 📡 실시간 접수 현황 헤더
- 총 신고 건수 · 오늘 접수 · 최다 사기 유형 실시간 표시

## 설치 및 실행

### 사전 요구사항
- **Node.js** 18.0 이상 — [nodejs.org](https://nodejs.org)
- **npm** 8.0 이상 (Node.js와 함께 설치됨)

### 1단계 — 저장소 복제 또는 다운로드

```bash
cd fraudsys
```

### 2단계 — 백엔드 의존성 설치

```bash
cd backend
npm install
```

### 3단계 — 프론트엔드 의존성 설치

```bash
cd ../frontend
npm install
```

### 4단계 — 테스트 QR코드 생성 (선택사항)

```bash
cd ../backend
node generate-test-qr.js
```

**방법 A: 원클릭 실행 (Windows)**

```
start.bat 더블클릭
```

## 사용 방법

### 신고 접수

1. http://localhost:5173 접속
2. **1단계**: 성명과 연락처 입력 (익명 가능 — 모든 항목 선택사항)
3. **2단계**: `📷 카메라로 QR코드 스캔` 버튼 클릭 → 의심 QR에 카메라 가져다 대기  
   또는 QR 내용을 텍스트 박스에 직접 붙여넣기
4. **3단계**: 증거 사진 업로드 (최대 4장 — 드래그 앤 드롭 또는 클릭)
5. **4단계**: `📍 현재 위치 자동 감지` 클릭 또는 주소 직접 입력
6. **5단계**: 사기 유형 선택 → 피해 상황 상세 기술 (최소 10자)
7. **6단계**: 내용 최종 확인 → `🚨 신고 제출하기`
8. 발급된 **접수번호 8자리**를 반드시 저장

### 신고 조회

1. http://localhost:5173/track 접속
2. 접수번호 8자리 입력 후 `조회하기`
3. 처리 현황 타임라인 및 신고 내용 전체 확인

### 관리자 대시보드

1. http://localhost:5173/admin 접속
2. 통계 카드 및 사기 유형 차트 확인
3. 검색창에 접수번호·신고자명·사기 유형·위치 등 입력하여 검색
4. 유형 드롭다운으로 필터링
5. 신고 항목 클릭 → 상세 내용 및 첨부 사진 확인
6. `CSV 내보내기` 버튼으로 전체 데이터 다운로드

## API 명세

### `POST /api/report`
신고서 제출

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| `reporterName` | string | 선택 | 신고자 성명 |
| `reporterContact` | string | 선택 | 연락처 |
| `fraudType` | string | 권장 | 사기 유형 |
| `qrData` | string | 선택 | QR코드 내용 |
| `locationText` | string | 선택 | 주소/장소명 |
| `latitude` | string | 선택 | GPS 위도 |
| `longitude` | string | 선택 | GPS 경도 |
| `description` | string | 권장 | 피해 상황 설명 |
| `images` | File[] | 선택 | 증거 사진 (최대 4장, 각 10MB 이하) |

**응답 예시**
```json
{
  "success": true,
  "reportId": "A1B2C3D4",
  "timestamp": "2024. 1. 15. 10:30:00"
}
```

---

### `GET /api/report/:id`
접수번호로 신고 조회

```
GET /api/report/A1B2C3D4
```

**응답 예시**
```json
{
  "success": true,
  "report": {
    "reportId": "A1B2C3D4",
    "timestamp": "2024. 1. 15. 10:30:00",
    "fraudType": "QR코드 사기",
    "locationText": "서울 강남구 테헤란로 152",
    "status": "검토 중",
    "statusCode": "reviewing"
  }
}
```

---

### `GET /api/reports/all`
전체 신고 목록 조회 (관리자용)

| 쿼리 파라미터 | 설명 | 기본값 |
|---------------|------|--------|
| `q` | 검색어 | — |
| `type` | 사기 유형 필터 | — |
| `limit` | 페이지당 건수 | 50 |
| `offset` | 시작 위치 | 0 |

```
GET /api/reports/all?q=강남&type=QR코드 사기&limit=10&offset=0
```

## 테스트 시나리오

테스트 전용 드라마 브리핑 페이지를 제공합니다.

```
http://localhost:3001/scenario
```

| 사건 | 피의자 | 혐의 | 피해 규모 |
|------|--------|------|-----------|

| 02 | 박도현 (29세) | GS25 홍대점 결제 QR 교체 | 870만원 |
| 03 | 강민서 조직 | 불법 투자 폰지 사기 | 23억원 |

```bash
cd backend
node generate-test-qr.js
```

시나리오 페이지에서 QR코드를 화면에 띄운 후, 신고 앱의 카메라 스캐너로 직접 스캔하여 전체 플로우를 테스트할 수 있습니다.

## 지원 사기 유형

| 유형 | 설명 |
|------|------|
| 보이스피싱 (Voice Phishing) | 전화·문자로 개인정보 및 금융정보 탈취 |
| QR코드 사기 | 가짜 QR코드로 피싱 사이트 유도 또는 결제 가로채기 |
| 스미싱 (Smishing) | 문자 메시지 링크를 통한 악성 앱 설치 유도 |
| 파밍 (Pharming) | 정상 사이트 접속 시 가짜 사이트로 자동 연결 |
| 대출 사기 | 허위 대출 광고로 수수료 편취 |
| 투자 사기 | 허위 수익률로 투자금 편취 (폰지 사기 포함) |
| 메신저 피싱 | 카카오톡 등 메신저로 지인 사칭 후 송금 요구 |
| 계좌이체 사기 | 물품 대금·렌트비 등 명목으로 이체 유도 |
| 가짜 쇼핑몰 사기 | 허위 쇼핑몰 개설 후 대금 편취 |
